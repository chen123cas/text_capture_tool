# Text Capture Tool

一个功能强大的全局文本捕获工具，能够自动监听鼠标选中的文本并保存到DOCX文档中。

## 功能特性

- **全局文本捕获**: 自动监听鼠标选中的文本，无需手动复制粘贴
- **智能标签**: 根据来源应用程序自动添加标签（如[网页]、[微信]、[Word]等）
- **自动保存**: 捕获的文本自动保存到DOCX文档
- **系统托盘**: 后台运行，不占用任务栏空间
- **配置灵活**: 可自定义捕获间隔、文本长度限制等参数

## 开发说明

本项目使用PyQt5构建图形界面，专注于文本捕获功能。主要技术特点：

- 基于PyQt5的现代化GUI界面
- 多线程处理，避免界面卡顿
- 完善的错误处理和日志记录
- 模块化设计，易于扩展

## 使用方法

1. **启动应用**: 运行程序后，会在系统托盘显示图标
2. **开始捕获**: 右键点击托盘图标，选择"开始捕获"
3. **正常使用**: 在任意应用程序中选中文本，工具会自动捕获
4. **查看结果**: 捕获的文本会自动保存到 `text_capture.docx` 文件
